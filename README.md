## 📖 Executive Overview
The Philippines faces intense challenges across transportation, digital literacy, governance, employment, healthcare, and agriculture. Information is decentralized, and citizens struggle to access reliable civic systems.

**UgnayPH** solves this by centralizing 6 critical solutions into a single, AI-powered web application. Featuring a custom **Cyber-Pinoy** glassmorphism design, it provides an accessible, extremely sleek, and unified experience for the Filipino people.

## 🎯 The 6 Modules (Addressing 9 SDGs)
1. **Smart Mobility (SDG 9 & 11)**: Helps commuters beat traffic with AI-powered route suggestions and crowd-sourced hazard reports.
2. **VibeCheck PH (SDG 4 & 16)**: Combats disinformation by scanning text, URLs, and images using Google Gemini AI to detect fake news.
3. **Governance Watch (SDG 16)**: Promotes transparency by allowing citizens to submit complaints, track projects, and upvote community concerns.
4. **Jobs & Skills (SDG 8)**: Connects Filipinos to employment opportunities using AI skill-matching.
5. **HealthReach (SDG 3 & 10)**: Provides remote symptom triage via AI and secure healthcare appointment booking.
6. **AgriLink PH (SDG 2 & 12)**: Empowers farmers with AI-driven crop advisories and local market trends.

## 🛠️ Technological Innovation
UgnayPH is a fully functional prototype built with modern, scalable technologies:
- **Google Gemini 1.5 Flash AI**: Powers deep content analyses, disinformation detection, and intelligent advisories natively within the platform.
- **Next.js 15 & TypeScript**: Delivers a lightning-fast, highly responsive frontend.
- **Supabase (PostgreSQL)**: Powers robust, real-time data storage and secure authentication.
- **Tailwind CSS**: Custom UI/UX featuring Philippine flag colors (Azure, Scarlet, Sun Gold) encapsulated in a premium glassmorphic aesthetic.

## 🚀 How to Test the Prototype
```bash
git clone https://github.com/luigibalingit14/UgnayPH.git
cd UgnayPH
npm install
npm run dev
```
*(Note: Rename `.env.example` to `.env.local` and insert the appropriate Gemini API and Supabase keys before launching.)*
